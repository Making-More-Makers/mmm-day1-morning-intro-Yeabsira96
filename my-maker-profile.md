### What do I want to make? | æˆ‘æƒ³åšä»€ä¹ˆï¼Ÿ

Pulse Link is an interactive hardware prototype that reads human emotional cues using multimodal sensing. The system will combine multiple sensors (heart rate sensors, temperature sensors, maybe even EDA - electrodermal activity sensors) to capture physiological signals that correlate with different emotional states. Using machine learning algorithms, the device will interpret these signals and provide real-time feedback about the user's emotional state. The prototype could be worn as a wearable device or used in an interactive installation setting. The goal is to create a system that not only measures emotions but also provides actionable insights to help users understand their emotional responses better.

### Why do I want to make this? | ä¸ºä»€ä¹ˆæƒ³åšè¿™ä¸ªï¼Ÿ

I want to make Pulse Link because it addresses a real problem in today's fast-paced world: many people struggle to understand and manage their emotions effectively. By creating a tool that makes emotions visible and quantifiable through data, we can help people become more emotionally self-aware. This project excites me because it sits at the intersection of hardware, AI/ML, and human-centered design - three areas I'm passionate about. Additionally, an emotional sensing system could have applications in mental health support, stress management, and personal wellbeing. Having this resume-worthy project will demonstrate my ability to integrate multiple technologies into a cohesive solution.
